# MShare

MShare — это бесплатная онлайн‑барахолка, где пользователи могут отдавать и находить вещи без оплаты.  
Сервис ориентирован на идею осознанного потребления: вещи получают вторую жизнь, а сообщество — простой и понятный инструмент обмена.

Каждый год миллионы предметов — от одежды и мебели до техники — отправляются на свалки.  
Многие из них всё ещё пригодны к использованию, но оказываются выброшенными.  
Это создаёт серьёзную экологическую проблему: растут полигоны отходов, загрязняется почва и вода, расходуются ресурсы, которые могли бы быть сохранены.

MShare помогает решать эту задачу. Вместо того чтобы выбрасывать, люди делятся вещами друг с другом.  
Так мы уменьшаем количество отходов, поддерживаем культуру повторного использования и создаём сообщество, где ценится забота о природе и друг о друге.

Наш проект объединяет людей, которые верят в ценность простых решений:  
- **Экономия ресурсов** — меньше производства новых вещей, больше повторного использования.  
- **Забота о планете** — снижение количества мусора и нагрузки на окружающую среду.  
- **Социальная поддержка** — возможность получить нужную вещь бесплатно, когда она действительно необходима.  
- **Удобство** — простой интерфейс, понятная навигация и возможность быстро разместить или найти объявление.  

MShare — это не просто сайт, это движение за осознанное потребление и экологичное будущее.  
Каждое объявление — маленький шаг к чистой планете и дружному сообществу, где ценится взаимопомощь.

## Функциональность
- Просмотр объявлений: Список карточек с названием, описанием и автором, аккуратно оформленных в едином стиле.
- Добавление вещей: Форма добавления с валидацией полей и уведомлением об успешном сохранении.
- Локальное хранение: Сохранение объявлений в LocalStorage для демонстрации работы данных на фронтенде.
- Навигация по разделам: Переход по страницам — Главная, Объявления, Добавить вещь, О проекте, Контакты.
- Единый дизайн: Общие шапка и футер, согласованные отступы, шрифты и цветовая палитра для визуального единства.
- Адаптация интерфейса: Базовая отзывчивость для корректного отображения на ноутбуках и десктопах (планируется расширение для мобильных).

## Основные функции
- Регистрация и вход пользователей
- Добавление объявлений «отдам даром»
- Поиск и фильтрация по категориям
- Возможность связаться с владельцем вещи

## Используемые технологии
- HTML, CSS
- JavaScript

## Запуск
1. Скачайте весь проект (папку MShare) на свой компьютер.  
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).  
3. Сайт запустится локально и будет доступен для просмотра.
