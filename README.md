# MShare

MShare — это бесплатная онлайн‑барахолка, где пользователи могут отдавать и находить вещи без оплаты.  
Проект создан в учебных целях.

## Основные функции
- Регистрация и вход пользователей
- Добавление объявлений «отдам даром»
- Поиск и фильтрация по категориям
- Возможность связаться с владельцем вещи

## Используемые технологии
- HTML, CSS
- JavaScript

## Запуск
1. Скачайте весь проект (папку MShare) на свой компьютер.  
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).  
3. Сайт запустится локально и будет доступен для просмотра.
